# CEFR Word Lists

This directory will contain word lists organized by CEFR levels (A1-C2).

## Format

Each language should have its own subdirectory with files named according to CEFR levels:

```
/data/cefr/
  /en/
    a1.txt
    a2.txt
    b1.txt
    b2.txt
    c1.txt
    c2.txt
  /es/
    ...
```

Each file should contain one word per line.
